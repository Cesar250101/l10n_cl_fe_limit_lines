Facturación Electrónica para Chile

Límite Líneas por Documento.

  • El uso de este módulo requiere de aprobar una certificación de facturador propio de mercado en el SII (O contar ya con ella) Para Mayor Información Click Aquí
  • El propósito del módulo, es establecer una estrategia para cuando se llena la cantidad máxima de líneas por Documento en formato A4 (debe ser todo en una hoja)
  • Este módulo es aplicable solamente sobre el Model account.invoice (Facturando)
  • Para el caso Inventario se creará uno especializado, ya que es opcional el uso de guías de despacho

Límite de líneas por DTE

Se puede configurar dos estrategias:

  • Detener Nuevas Lías
  • Crear Nuevo Documento al momento de validar

Detener nuevas líneas:

Para este caso, cuando se llegue al máximo permitido, el usuario no podrá crear más líneas

Crear Nuevo Documento al validar:

Para este caso, el usuario podrá crear más líneas del máximo establecido, sin embargo, cuando llegue valide el documento, se crearan 2, 1 hasta al máximo de líneas y otro con el resto de lías de la factura original. El codigo copiará tal cual el documento de origen, luego distribuirá las líneas según orden establecido antes de validar el documento
